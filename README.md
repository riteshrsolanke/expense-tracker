Django Expense Tracker
Overview

Django Expense Tracker is a full-stack web application for managing personal finances. Users can add, delete, and view expenses, filter them by month, and visualize spending patterns through interactive charts. The app includes a secure login/signup system and a Django admin panel for managing all users’ data.

This project is also data science ready, as all expense data is structured and can be used for analytics, forecasting, and trend analysis.

Features

User authentication: Signup, Login, Logout

Add expenses with amount, reason, and date

Delete and view past expenses

Filter expenses by month and year

Interactive bar charts of monthly expenses using Chart.js

Django admin panel to manage users and expenses

Ready for data science analysis (future enhancement)

Tech Stack

Backend: Django

Frontend: HTML, CSS (Bootstrap), Chart.js

Database: SQLite (default)

Authentication: Django User model
